# Web API directory
```
i/
```

## Data (backup)

#### Pull all data (export backup)
```
i/pull
```

#### Push data (import backup)
```
i/push
```

## User

#### pull all users
```
i/user/pull
```

#### pull params
```
i/user/pull?name=$
```

#### push user
```
i/user/push?name$
```

## Provider

Providers (Aka fornecedores) are owners of products

#### pull all providers
```
i/provider/pull
```

#### pull  params
```
i/provider/pull?name=$
```

#### push provider
```
i/provider/push?name$
```

## Product

#### pull all providers
```
i/product/pull
```

#### pull  params
```
i/product/pull?name=$
```

#### push product
```
i/product/push?name$&code=$&provider_id=$&category_id=$
```
