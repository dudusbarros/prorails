# Web Api directory
```
i/
```

## Data (backup)

#### Pull all data (export backup)
```
i/pull
```

#### Push data (import backup)
```
i/push
```

## User

#### pull all users
```
i/user/pull
```

#### pull params
```
i/user/pull?name=$
```

#### push user
```
i/user/push?name$
```

## Provider

#### pull all providers
```
i/provider/pull
```

#### pull  params
```
i/provider/pull?name=$
```

#### push provider
```
i/provider/push?name$
```
